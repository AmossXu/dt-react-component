# dt-react-component

## 数栈前端组件管理库
### 开发
```
$ git clone git@github.com:zhangtengjin/dt-react-component.git

$ npm i
$ # 开发
$ npm run storybook
$ # 部署构建
$ npm run build-storybook

```
